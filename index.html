You may have known this game by it's past name "Turbulence", a product of team WAW. Now it's back, but with more features, and some more of that trademarked team FLR beary goodness.

5 easy steps to follow

 

Step 1. Play the Game

Download the Game

Download the appropriate .jar file for your operating system from the selection below

Windows .jar

 

Install the game

Turbearlence requires no installation. Simply double click on the .jar file you downloaded to begin the game.

 

Notice all of the nice new features

  1. An Airport you can land and take off from
  2. 10 planes in the airspace
  3. Planes follow their flight plan automatically (manual control can be used to avoid collisions)
  4. User scoring system

Need help?

If you need any help with installing the game then feel free to contact any of the team members. You can find our details on the contact us page.

If you need help playing the game then click here to download our user manual.

Step 3. Read the reports

All of the changes we have made to make Turbulence into Turbearlence are thoroughly documented.

Extension Report

This report outlines in depth each of the changes that we made to the game. It includes justification for each change and a description of how it was achieved. Each change also includes a reference to the relevant section of the testing report so you can see exactly how it was tested.
